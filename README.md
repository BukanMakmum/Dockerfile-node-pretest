# Dockerfile node.js


build image
```
$ docker build -t pretest-image .
```

run image:
```
$ docker run pretest-image
```

run container:
```
$ docker run --name MyContainer -it pretest-image bash
```
uji melalui terminal:
```
curl -v localhost:3000
```
atau uji melalui browser, akses:
```
http://localhost:3000
```
